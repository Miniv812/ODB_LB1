# Лабораторна робота 1: Збір вимог та розробка схеми ER
Наша задача розробити БД для гри (з можливістю розширення).
 За сутность візьмемо основне, що є в грі, це: Персонаж, Гільдія, Предмети, Зони, Квести та Чудовиська. 
Для створення персонажа нам потрібна реальна людина, тому додаємо ще сутність Юзера. 
Придумаємо атрибути для цих сутностей та певні зв'язки між ними та проектуємо усі наші рішення на єкран.
